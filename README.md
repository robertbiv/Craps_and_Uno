# Craps and Uno Card Games

JavaScript implementations of classic Craps dice game and Uno card game with interactive web-based interfaces featuring game logic, scoring systems, and player interactions.

## Overview

This repository contains two popular game implementations built with vanilla JavaScript and HTML:

- **Craps**: A dice game simulation that follows traditional casino Craps rules with pass line betting, odds calculation, and multi-round gameplay
- **Uno**: A digital version of the classic card game featuring color matching, special action cards, and multiplayer functionality

## Features

### Craps Game
- Authentic casino-style Craps gameplay
- Pass line and come out roll mechanics
- Point establishment and resolution
- Win/loss tracking and scoring system
- Interactive dice rolling with visual feedback

### Uno Game
- Full deck of Uno cards with all colors and special cards
- Draw 2, Skip, Reverse, and Wild card functionality
- Turn-based gameplay logic
- Card matching rules enforcement
- Player hand management
- Win condition detection

## Technology Stack

- **JavaScript (89.5%)**: Core game logic, event handling, and state management
- **HTML (10.5%)**: User interface structure and layout
- Pure vanilla JavaScript - no frameworks or libraries required

## How to Use

1. Clone or download this repository
2. Navigate to either the `Craps` or `Uno` folder
3. Open the `index.html` file in any modern web browser
4. Follow the on-screen instructions to play

No build process or dependencies required - just open and play!
